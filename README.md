# 🚀 AI Student Productivity System

An AI-powered system to predict student performance and generate personalized study improvement plans.

---

## 🎯 Features

- 🤖 **ML Predictor**
  - Predicts student final grade
  - Uses Random Forest model

- 🧠 **AI Insights (Gemini)**
  - Weak area detection
  - Personalized study plan
  - Smart productivity tips

- 📊 **Analytics Ready**
  - Expandable for dashboards

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas / NumPy
- Google Gemini API

---

## 🧠 How It Works

1. User inputs student data  
2. ML model predicts performance  
3. Gemini AI generates improvement plan
